Projects

NLP Next Word Prediction project, I engaged in extensive data preprocessing and feature engineering, addressing challenges such as handling large text files and ensuring data quality by removing unnecessary whitespace, non-English words,and convert to lowercase text. I compared two next-word prediction methods: Markov Models and n-grams, and applied machine learning techniques, such as bagging, to manage data complexity effectively. My efforts culminated in the development of a predictive model that achieved a 90% accuracy rate, demonstrating my ability to transform raw data into actionable insights. Additionally, I created a user-friendly Shiny app that allows users to visualize prediction results in real-time.

Object Detection in Images Using Deep Learning with YOLO project involved creating a robust object detection system using the YOLO (You Only Look Once) algorithm. I managed the end-to-end process, including collecting and preprocessing large image datasets, addressing challenges related to dataset variability and ensuring model accuracy. I focused on developing a model capable of detecting and classifying objects within images with high precision, which is essential for real-time applications like autonomous vehicles, surveillance systems, and robotics. The project leveraged deep learning techniques and Python for implementation, and the skills honed included advanced image processing and efficient handling of Linux environments.

Clustering Methods for Anomaly Detection in Big Data the project involved developing a sophisticated system for detecting outliers in large datasets using various clustering techniques, including K-Means, DBSCAN, Hierarchical Clustering, and Isolation Forest. I tackled challenges related to managing the complexity of big data by employing dimensionality reduction methods such as hashing, which enhanced processing efficiency and scalability. By optimizing the algorithms, I improved the system’s accuracy and performance, enabling effective anomaly detection for critical applications like fraud detection and network security. The project demonstrated my ability to handle large-scale data and apply machine learning techniques to deliver actionable insights.

An Ensemble Angle-Based Outlier Detection for Big Data, I addressed the challenge of outlier detection in high-dimensional and large-scale datasets. Traditional outlier detection methods struggle with the vast distance between points in such datasets, leading to inaccurate results. To overcome this, I developed an ensemble method that combines angle-based outlier detection with dimensionality reduction techniques. By applying Locality Sensitive Hashing (LSH) for data reduction and leveraging ensemble techniques, I improved the efficiency and accuracy of outlier detection. I further enhanced the method by using Support Vector Machines (SVM) to aggregate results from various subsets. Through testing with real-world datasets, my approach demonstrated significant improvements in identifying outliers effectively.
